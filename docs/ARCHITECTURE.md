# Floki-v2 Architecture

## Overview

Floki-v2 follows a brain-modular architecture inspired by human brain regions. Each module represents a specific brain region with dedicated functionality.

## Brain Modules

- **Amygdala**: Emotion processing (scaffold only)
- **Broca**: Language processing (scaffold only)
- **Cerebellum**: Motor control and coordination (scaffold only)
- **Emotions Base**: Core emotion system (scaffold only)
- **Frontal**: Executive functions and decision making (scaffold only)
- **Hippocampus**: Memory formation and retrieval (scaffold only)
- **Occipital**: Visual processing (scaffold only)
- **Temporal**: Auditory processing and language comprehension (scaffold only)
- **Thalamus**: Sensory relay system (scaffold only)
- **Personality**: Personality traits and expression (scaffold only)
- **Pineal**: Circadian rhythm and sleep regulation (scaffold only)

## Current Status

All modules are currently in scaffold mode with no production implementation.
