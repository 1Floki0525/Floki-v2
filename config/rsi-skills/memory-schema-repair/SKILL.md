# Memory schema repair

Fix structured memory records (engineering/personal) and durable learning writes.

## When to use
A cycle must write a durable learning record, or a memory record is malformed.

## Rules
- Keep personal continuity and engineering memory separate.
- Every approved/denied/failed/no-candidate cycle creates a structured durable record.
- Retrieve compact task-relevant capsules; do not inject all memory into every cycle.
